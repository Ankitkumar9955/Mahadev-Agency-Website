# Mahadev Agencies – Solar Installation Website

A premium, modern, and fully responsive website developed for **Mahadev Agencies**, a rooftop solar installation and distribution firm based in Odisha, India.

The website is designed to provide homeowners and businesses with an intuitive platform to explore solar solutions, government subsidy programs, and request installation quotes.

---

## Features

- Modern premium landing page
- Responsive design across all devices
- Interactive UI with smooth animations
- Government subsidy information
- Service showcase
- Lead generation contact section
- Clean navigation
- Performance optimized
- SEO-friendly structure

---

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Responsive Design

---

## Project Overview

The primary objective was to build a visually engaging marketing website that communicates trust, professionalism, and simplicity while helping potential customers understand rooftop solar adoption and government subsidy programs.

The design follows a modern product-focused aesthetic inspired by Apple, Vercel, and Linear, emphasizing clean layouts, elegant typography, immersive visuals, and intuitive user experience.
